Game of Life

Type de projet Personnel
Applications utilisées Visual Studio Code
Langages utilisés Python
Période Fais en 3 jours en 2021
Le jeu de la vie est une simulation en 2D créée par John Horton Conway, où des règles très simples permettent de générer une simulation visuelle rappelant certains aspects de la vie. J'ai réalisé ce projet lorsque j'étais en terminale, car je trouvais cette simulation très intéressante et je voulais mettre à l'épreuve mes connaissances en langage Python.
  • J'ai dû apprendre à utiliser Pygame, une bibliothèque Python permettant de créer des fenêtres visuelles et principalement utilisée pour le développement de jeux en 2D. Cependant, je l'ai seulement utilisée pour afficher des cases noires ou blanches et pour récupérer les entrées clavier.
  • Pour recréer la simulation j'ai dû comprendre les règles du jeu de la vie puis ensuite les recoder en Python. Cela m'a beaucoup appris, car j'ai dû concevoir un algorithme fonctionnel ainsi qu'à réfléchir à une architecture de code.

Motus

Type de projet Personnel
Applications utilisées Visual Studio Code
Langage Utilisé Python
Période Mai 2022 (moins d'une semaine)
Ce projet est une reproduction d'une émission de télévision française appelée Motus, dans laquelle les candidats doivent deviner un mot en utilisant uniquement les lettres qu'il contient.
  • J'ai utilisé la bibliothèque Pygame pour avoir une fenêtre et y afficher des images. J'ai donc créé une classe Image pour pouvoir changer les images et définir leur taille dans la fenêtre.
  • J'ai programmé l'interface utilisateur de manière à ce qu'elle s'adapte à la taille de n'importe quel écran.
  • J'ai programmé un code qui choisi un mot au hasard dans un fichier texte donné contenant une liste de mots. J'ai également créé un code qui a modifié ce fichier texte de manière à ce que chaque mot ne contienne pas de caractères "-" ni d'accents sur les lettres.
